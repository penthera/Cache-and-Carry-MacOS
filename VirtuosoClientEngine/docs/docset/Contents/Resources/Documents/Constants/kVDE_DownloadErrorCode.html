Constants

kVDE_ServerFileSizeDisagreesWithExpectedFileSize

The file length advertised by the HTTP server doesn’t match the expected size. If file size validation is off, then the download proceeds anyway. If it’s on, Virtuoso aborts the download and sets the appropriate error codes.

Declared In VirtuosoConstants.h.

kVDE_FileLengthMismatch

When the download completed, the observed final size didn’t match the expected size. If file size validation is on, Virtuoso deletes the file and retries the download. If it’s off, Virtuoso marks the download complete and successful.

Declared In VirtuosoConstants.h.

kVDE_FileMD5Mismatch

If you supplied an expected MD5 for the file AND the observed MD5 didn’t match, then Virtuoso deletes the file and restarts the download.

Declared In VirtuosoConstants.h.

kVDE_InvalidMimeType

If the MIME type returned from the HTTP server doesn’t match the required MIME types whitelist you provided earlier, then Virtuoso aborts the download.

Declared In VirtuosoConstants.h.

kVDE_NetworkError

Some network issue (HTTP 404,416,etc) caused the download to fail.

Declared In VirtuosoConstants.h.

kVDE_FileSystemError
